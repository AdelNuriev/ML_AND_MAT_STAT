# Отчет по задаче: Классификация рукописных цифр (dataset digits)

## 1. Постановка задачи

### 1.1. Цель работы
Определить число на картинке из датасета `digits` (рукописные цифры от 0 до 9) с помощью методов кластеризации и классификации библиотеки `sklearn`.

**Ограничения:**
❌ Запрещено использовать сторонние библиотеки (OpenCV и др.)
✅ Разрешены только `sklearn`, `numpy`, `matplotlib`, `pandas`

### 1.2. Основные этапы
1. Разбить датасет `digits` на обучающую (85%) и тестовую (15%) выборки.
2. Найти лучший классификатор из минимум 3-х с подбором гиперпараметров.
3. Визуализировать все примеры, где лучший классификатор ошибся.
4. Сравнить результат с классификацией после предобработки `t-SNE`.

### 1.3. Характеристики датасета
- **Количество изображений:** 1797
- **Размер каждого изображения:** 8×8 пикселей (64 признака)
- **Количество классов:** 10 (цифры 0–9)
- **Целевая переменная:** цифра (0–9)

Файл с решением: Recognizing_numbers.py

## 2. Предобработка данных

### 2.1. Загрузка и разделение

```python
from sklearn.datasets import load_digits
from sklearn.model_selection import train_test_split

digits = load_digits()
X, y = digits.data, digits.target

# Разделение: 85% train, 15% test
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.15, random_state=42, stratify=y
)
```
